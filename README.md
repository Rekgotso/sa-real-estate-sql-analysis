# SA Real Estate SQL Project

This project provides a structured SQL-based analysis of the South African real estate market, including schema setup, sample data, analytical queries, and visualizations.

## Folder Structure

```
sa-real-estate-sql/
├── README.md
├── schema/
│   ├── create_tables.sql
│   └── create_indexes.sql
├── data/
│   ├── sample_properties.csv
│   ├── sample_agents.csv
│   └── data_generation_script.py
├── queries/
│   ├── 01_market_overview.sql
│   ├── 02_sales_performance.sql
│   ├── 03_feature_impact.sql
│   ├── 04_geographic_trends.sql
│   ├── 05_price_volatility.sql

│   ├── 08_size_analysis.sql
│   ├── 09_investment_opportunities.sql
│   └── 10_comprehensive_report.sql
└── visualizations/
    └── query_results_viz.ipynb
```

## Purpose
This repository provides everything required to run SQL-driven real estate market analysis including:
- Database schema creation
- Sample datasets
- Modular SQL queries for insights
- Jupyter Notebook for visualizations

## Getting Started

1. Run scripts in `/schema` to create database schema.
2. Load sample CSV data into your database.
3. Execute queries in `/queries`.
4. Use the notebook in `/visualizations` for visual exploration.
